針對鈺創科技六種產品的歷史需求量資料，設計一套時間序列的三步預測方法。其中我們嘗試了以下模型：
1. 傳統時序模型 ARIMA
2. 機器學習模型 Random Forests, XGboost
3. 集成式學習 Ensemble learning

以及分別使用直接和間接兩種三步預測的方法，並藉由 rMSE 指標比較各種模型和方法之間的優劣勢。
