以 Rshiny 套件建立一個互動式介面，為輸入的二維資料藉由 Gaussian Process model 預測補值，提供使用者選擇 covariance family 及各種參數的調整，再將預測結果以二維熱點圖呈現。

(How to apply : 下載 110024516_GP_demo.R -> click Run App -> 匯入 train.csv)
